웹프로그래밍 프로젝트

2022.05.27

안녕하세요. 컴퓨터공학부 임은상입니다.
이번 프로젝트는 보시는 바와 같이 블로그 형식으로 구현해봤어요. 실제 블로그 포스팅하듯 글을 작성했습니다.

저희 중간고사 보기 이전에 실습할 때 카카오맵 API 사용하여 화면에 출력했던 거 기억하시나요? 웹 제작을 하다 보면 앞으로 자주 쓰일 거 같아 좋은 주제라는 생각이 문뜩 들었어요. 따라서 이번 프로젝트에서 소개할 저의 첫 번째 콘텐츠는 '카카오맵 API 응용하기' 입니다. 제목 그대로 여러 기능을 추가하여 응용해보았어요. 또한 자바스크립트 언어로 구현했기 때문에 사실 이 콘텐츠 속 서브 컨텐츠는 'javascript 사용하기' 라고도 할 수도 있겠네요😎 카카오맵 API 공식 사이트를 참고하여 진행했어요. 콘텐츠에 대한 개요 설명은 여기까지 하고 페이지로 이동해서 상세한 설명을 진행해보겠습니다.

다음 콘텐츠는 '계산기 응용하기 시리즈' 입니다. 수업에서 서블릿과 JSP를 실습하면서 계산기를 구현했었습니다. 이 계산기 응용하기 콘텐츠에서는 실습에서 구현했던 계산기를 좀 더 계산기같게 업그레이드 시켰습니다. 그렇다고 거창한 기능을 추가한 건 아니에요😁 상세 페이지에서 수업에서 다뤘던 내용을 중점으로 어떤 방식으로 프로그램이 동작하는지 설명하겠습니다.

저는 수업에서 다룬 모든 내용을 포함하기보다 이렇게 실습 내용을 위주로 프로젝트를 진행해봤어요. 실습 내용 구현에 대한 설명을 하다보면 배운 개념들도 자연스럽게 포함되므로 매우 효율적이라고 생각했어요.
긴 글 읽어주셔서 감사합니다. 제 블로그에 또 놀러오세요.
