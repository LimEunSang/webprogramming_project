안녕하세요. 컴퓨터공학부 임은상입니다. 지금 이 글을 읽고 있는 학우 여러분들 프로젝트 하시느라 고생하셨습니다!
이번 프로젝트는 보시는 바와 같이 블로그 형식으로 구현해봤어요. 컨셉에 맞게 블로그 포스팅할 때 흔하게 볼 수 있는 그런 말투를 사용할 예정이니 양해 부탁드려요😊

저희 중간고사 보기 이전에 실습할 때 카카오맵 API 사용하여 화면에 출력했던거 기억하시나요? 웹 제작을 하다보면 앞으로 자주 쓰일 거 같아 좋은 주제라는 생각이 문뜩 들었어요. 따라서 이번 프로젝트에서 소개할 저의 첫 번째 콘텐츠는 '카카오맵 API 응용하기' 입니다! 제목 그대로 여러 기능을 추가하여 응용해보았어요. 또한 자바스크립트 언어로 구현했기 때문에 사실 이 컨텐츠 속 서브 컨텐츠는 'javascript 사용하기' 라고도 할 수도 있겠네요😎 카카오맵 API 공식 사이트를 참고하여 진행했어요. 컨텐츠에 대한 개요 설명은 여기까지 하고 페이지로 이동해서 상세한 설명을 진행해보겠습니다.
